# Usage

```cmd
C:\> yarn init -y
C:\> yarn add @commitlint/config-conventional @commitlint/cli -D
C:\> echo module.exports = {extends: ['@commitlint/config-conventional']} > commitlint.config.js
C:\> yarn add husky -D
C:\> 
```

* OBS: Required .git folder in project
